13.2.15 _unlink_r—Reentrant version of unlink

Synopsis 

#include <reent.h>
int _unlink_r(struct _reent *ptr, const char *file);

Description
This is a reentrant version of unlink. It takes a pointer to the global data block, which holds errno.
