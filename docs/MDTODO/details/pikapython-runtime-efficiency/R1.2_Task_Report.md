# R1.2 dict/list 热点报告

- 已实施：
  - `PikaStdData_Dict_get()` 复用第一次 `pikaDict_get()` 的结果，命中路径由两次链表查找降为一次。
  - 同步修改源码目录和 Linux package 副本。
- 未实施：
  - dict/list 保持单链表，正序 O(n) 遍历无法同时满足零额外 RAM、顺序语义和重入安全。
  - 临时反转会使回调期间容器物理顺序变化；反向指针、tail 或游标缓存会增加对象 RAM；因此本轮不采用。
  - 现有 move-to-front 代码没有正确摘除原节点，继续保持关闭。
- 验证：
  - dict get、items、items2、items_kv 和关键字字典测试通过。
  - 默认与 minimal core 的 `data/bss` 均无增长。
