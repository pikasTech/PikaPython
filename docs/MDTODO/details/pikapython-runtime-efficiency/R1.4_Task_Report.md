# R1.4 VM yield 优化报告

- 修改：
  - `PIKA_EVENT_ENABLE=0` 时不再编译每指令 yield 判断与空函数调用。
  - event 启用且 period 为 1 时直接调用 yield，避免运行时取模表达式。
  - period 大于 1 时保留原周期判断，调度语义不变。
- 验证：
  - minimal/event-off 隔离全量回归 178/178 通过。
  - minimal core 的 `text` 从 1022833 降至 1022725，减少 108 字节。
  - minimal core 的 `data=47180`、`bss=23074`，均保持不变。
