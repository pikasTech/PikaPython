# R1.5 验证报告

- 行为回归：
  - 定向关键字、默认参数、parser、dict items：8/8。
  - default、float、optimize-speed、pool 相关 suites：各 69/69。
  - minimal 相关过滤：1/1；minimal 隔离全量：178/178。
  - 默认配置隔离 LVGL、hashlib 与 network/socket/mqtt/requests/jrpc 后：387/387。
- 资源：
  - 默认 core：`text 1072241 -> 1072197`，减少 44 字节；`data/bss` 不变。
  - minimal core：`text 1022833 -> 1022725`，减少 108 字节；`data/bss` 不变。
  - 默认完整测试二进制尺寸完全相同。
- 性能：
  - 三轮交替测量中 `function_call_1000` 原始中位数分别改善约 7.0%、0.6%、2.0%。
  - 同轮 while 控制也有明显波动，不能把该墙钟结果视为稳定 MCU 提速比例。
  - 确定收益是每次参数绑定减少两次 allocator 调用、参数扫描 O(n²) 降为 O(n)、关键字 reverse O(k²) 降为 O(k)。
- 隔离项：
  - LVGL 子模块上游不可匿名拉取。
  - hashlib 与 NC01 mbedTLS 3 API 不兼容。
  - 网络类测试按用户要求隔离。
