# R1.3 任务报告

- `#347` 根因：
  - 可变位置参数边界只计算必选位置参数。
  - 默认参数尚未消费时，传入值被过早放入 `*args`。
  - 修复边界为 `n_positional + n_default`。
- `#348` 根因：
  - `Arg` 仅保存参数名哈希。
  - 构造 `**kwargs` 字典时直接把哈希十进制文本当作可见键。
  - 最小方案是在当前字节码常量池中按既有哈希恢复原始关键字文本，失败时保留旧回退。
- `#322` 根因：
  - 多目标 `for` 拆分对子 AST 和子 ASM 的失败结果缺少检查。
  - `NULL` 先进入字符串拼接断言，随后空 AST 又进入析构。
  - 最小方案只传播已有解析失败，不实现新语法。
