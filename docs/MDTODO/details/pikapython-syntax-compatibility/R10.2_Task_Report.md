# R10.2 任务报告

初始 24 项探针发现多项错误接受，并在 `x = [i for in values]` 上触发 `strGetSize(NULL)` 断言、中止进程（退出 134）。根因是推导式 helper 无条件保存可能为 NULL 的 `for/in` 片段，代码生成阶段再解引用。

本批选定 15 项：明显非法左值、多个星号 list 解包、缺失 for/in 片段、if/while 重复顶层冒号、字典缺逗号和四类推导式缺项。剩余 9 项为 orphan else/elif/except/finally、循环/函数外 break/continue/return、list/tuple 缺逗号，涉及 block 上下文或通用 token 邻接，移交后续批次。
