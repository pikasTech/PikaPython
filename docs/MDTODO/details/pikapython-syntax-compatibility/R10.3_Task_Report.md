# R10.3 任务报告

实现保持在 `PikaParser.c` 现有分层内：

- 推导式 helper 改为返回 `PIKA_RES`，在写 AST 属性前验证表达式、目标、`in` 和 iterable 均存在，消除 NULL 解引用。

- `if/elif/while` 要求恰好一个顶层冒号。
- 新增 15 项表驱动非法矩阵及普通变量、tuple 解包、下标赋值、有效推导式邻接测试。

未修改 VM/runtime 热路径、AST 架构或语法支持范围。
