# R11.2 任务报告

## 纳入 20 项模板的解析问题

1. 孤立 `else:`：CPython 编译报错，Pika 返回 0。
2. 孤立 `elif True:`：CPython 编译报错，Pika 返回 0。
3. 孤立 `except:`：CPython 编译报错，Pika 返回 0。
4. 孤立 `finally:`：CPython 编译报错，Pika 返回 0。
5. 循环外 `break`：CPython 报 `'break' outside loop`，Pika 返回 0。
6. 循环外 `continue`：CPython 报错，Pika 返回 0。
7. 函数外 `return 1`：CPython 报错，Pika 返回 0。
8. 函数外 `yield 1`：CPython 报错，Pika 返回 0。
9. list 项缺逗号 `x = [1 2]`：CPython 报错，Pika 返回 0。
10. 空 `import`：CPython 报错，Pika 在 `Suger_import()` 触发 SIGSEGV。
11. `from x import` 缺名称：CPython 报错，Pika 在 `Suger_import()` 触发 SIGSEGV。

## 附加发现

另有 16 项错误接受未计入 20 项模板，包括 tuple 缺逗号、模块级 nonlocal、删除 literal、非法 class/def 名、非法 for 左值、空 assert、非法 lambda/条件表达式、过多 slice 冒号、`**kwargs` 后形参、重复 varargs、tuple 增强赋值、literal 注解、函数外 await 和 async for。
