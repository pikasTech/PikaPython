# R11.3 任务报告

## 纳入 20 项模板的运行时问题

12. 默认参数求值时机错误：`seed=1; def f(a=seed); seed=2; f()` 应为 1，实际为 2；可变默认值未跨调用保留，副作用默认表达式也被每次调用重复执行（期望汇总 111，实际 212）。三个表现归为同一合同。
13. 幂运算错误地左结合：`2 ** 3 ** 2` 应为 512，实际为 64。
14. `and` 不短路：`False and fail()` 仍执行 `fail()`，产生 NameError，`result` 未生成。
15. `or` 不短路：`True or fail()` 仍执行 `fail()`，产生 NameError，`result` 未生成。
16. list 推导式破坏外层变量作用域：`i=9; [i for i in [1,2]]; result=i` 应保留 9，实际 `result` 无法作为整数读取（-999999999）。
17. 星号解包赋值编译成功但运行失败：`a,*b=[1,2,3]` 应得到 `b=[2,3]`，实际 NameError: `b` 未定义。
18. 解包值过多未报错：`a,b=[1,2,3]` 应抛 ValueError，实际继续执行并得到 `result=1`。
19. 解包值不足错误类型不兼容：`a,b,c=[1,2]` 应抛 ValueError，实际报告 IndexError 且 `result` 未生成。
20. 未绑定局部变量错误缺失：函数内先读取后赋值同名变量应抛 UnboundLocalError，实际读取全局值并得到 `result=1`。
