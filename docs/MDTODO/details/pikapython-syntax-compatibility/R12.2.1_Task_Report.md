# R12.2.1 任务报告

修复 10 个 parser 错误接受：孤立 `else`、`elif`、`except`、`finally`，循环外 `break`/`continue`，函数外 `return`/`yield`，以及 list/tuple 内相邻数值 literal 缺逗号。

实现复用现有 block 栈，不新增持久 parser 状态：当前行退栈时记录直接前驱以校验 clause；从栈顶向外查找 loop/def 上下文，并以嵌套 class/def 作为作用域边界；数值缺逗号复用既有 token 邻接扫描。改动只在 parser 编译路径，不触及 VM/runtime 热路径。

隔离 Linux 树同步 core 并重建后，新增非法/合法邻居测试、R6/R8/R10 parser 回归共 11/11 通过；现有 `except.try_while_return` 运行测试通过。其余附加 parser 候选已拆入 R12.2.2-R12.2.3，父任务保持进行中。
