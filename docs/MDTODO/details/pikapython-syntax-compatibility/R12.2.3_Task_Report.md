# R12.2.3 任务报告

修复附加表达式候选中的 5 个错误接受：非法空体 lambda、缺条件的条件表达式、超过两个冒号的 slice、函数外 await 和 async for。上述语法不在当前 PikaPython 子集内，本轮明确返回语法错误，不扩展实现范围。

实现仅在 RHS AST 入口、slice 项计数和行级 statement 入口增加局部校验；字符串中的同名文本、普通 `await_result` 标识符、三段 slice 和普通 for 保持可编译。首次回归发现 lambda 校验在赋值拆分前执行，已移到 RHS 拆分后并补 `lambda:` 边界。

隔离 Linux 树同步 core 并重建后，新增非法/合法邻居通过，完整 parser 测试加现有异常运行测试共 16/16 通过。
