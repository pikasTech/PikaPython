# R14 任务报告

R14 在已有语法子集和现有 VM 架构内完成一轮函数参数绑定 runtime 复核与热路径优化。

- #347-#349 的默认参数、*args、**kwargs 和必选 keyword 行为均有 Linux 回归，新增组合邻接用例。
- 参数 typelist 从三次扫描收敛为一次，并删除第二个 256 B 临时缓冲和相关复制、反向扫描。
- default 36/36、minimal 225/225 通过；default 静态尺寸不变，minimal text 减少 64 B，静态 data/bss 不变。
- A/B 受宿主机噪声影响，没有宣称 CPU 加速；确定收益是扫描次数减少和每次相关调用少用 256 B 临时堆。
- kwargs 名称恢复的常量池线性扫描仍存在，但消除它需要新增元数据或缓存，超出本轮非架构、低 RAM 边界。
- #347、#348、#349 已回写证据，等待修复合并后再关闭。
