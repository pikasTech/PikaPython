# R5.2 任务报告

## 修复

- 将完整配置的 `_kw_to_pos_all()` 从断言式 `void` 改为成功/失败返回值。
- keyword 无法补齐固定形参时返回 `pika_false`，调用方设置 `PIKA_RES_ERR_INVALID_PARAM` 和可捕获 `TypeError: invalid keyword args`。

新增 #328 原生方法回归，覆盖位置、混合、乱序全 keyword、缺参和未知 keyword 可捕获错误。重复位置/keyword 赋值仍可能被重排而未报错，作为 #328 后续 Python 3 精确语义问题保留，本轮不以 Flash 增长扩大修复。
