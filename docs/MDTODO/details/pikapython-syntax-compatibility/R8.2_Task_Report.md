# R8.2 任务报告

## 语义校验

- `_defGetDefault()` 记录参数名、默认值顺序和 keyword-only 分界。
- method AST 记录 keyword 名称和 keyword 后的位置参数状态。
- dict AST 要求每个非空项只有一个顶层冒号。
- 保留空参数、尾逗号和字典尾逗号等合法邻接行为。

## 崩溃修复

- 根因是子 AST 失败时立即释放对象，父队列仍持有悬空指针。
- `AST_parseStmt()` 失败后只返回 `NULL`，不再抢先释放子对象。
- 子表达式逐层传播 `PIKA_RES_ERR_SYNTAX_ERROR`。
- 最外层 AST 所有者统一释放整棵树。
- 不改变 AST 数据结构、parser 架构或 VM/runtime 热路径。
